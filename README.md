# Coding Judge Platform

A scalable online coding judge platform built using Spring Boot Microservices.

## Architecture

- Eureka Discovery Server
- API Gateway
- Problem Service
- Submission Service (In Progress)
- Authentication Service (Planned)

## Tech Stack

- Java
- Spring Boot
- Spring Cloud
- Spring Data JPA
- MySQL
- Docker
- RabbitMQ
- Redis

## Features

### Completed
- Eureka Service Discovery
- API Gateway
- Problem Management Service
- Global Exception Handling

### In Progress
- Submission Service
- Code Execution Engine

### Planned
- JWT Authentication
- Docker Deployment
- Redis Caching
- RabbitMQ Messaging
